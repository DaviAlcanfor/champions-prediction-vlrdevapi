## Data Sources

This project uses the following public datasets from Kaggle:

- [Valorant VCT 2025 - All Events](https://www.kaggle.com/datasets/piyush86kumar/valorant-vct-2025-all-events) by piyush86kumar
- [Valorant Champion Tour 2021-2025](https://www.kaggle.com/datasets/ryanluong1/valorant-champion-tour-2021-2023-data) by ryanluong1

Data is not included in this repository. Download the datasets and place them in `data/raw/` following the structure in `src/ingestion/kaggle_loader.py`.
